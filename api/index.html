POST MemberRegister.php
Request the MemberRegister.php after doing the auth reuqests from Facebook API
Parameters
  • FacebookId: Get the Facebook ID from Facebook Graph API
  • Name: Get Name from Facebook
  • Email: Get the Email address from Facebook Graph API
  • Example: MemberRegister.php?FacebookId=123&Email=test@gmail.com&Name=Cooltey
    • Return { "StatusCode": "200", // Status Code (200, 404) "MemberId": "123" // Member Id }
    POST MemberLogin.php
    Send Facebook ID and Email address to verify account information, and then return the member id
    Parameters
  • FacebookId: Get the Facebook ID from Facebook Graph API
  • Email: Get the Email address from Facebook Graph API
  • Example: ./MemberLogin.php?FacebookId=123&Email=test@gmail.com&MemberId=1
    • Return { "StatusCode": "200", // Status Code (200, 404) "MemberId": "123" // Member Id "LoginToken": "RANDOMSTRING" // Member Id }
    POST BrowserAdd.php
    Simply add browse history, it is for the Machine Learning
    Parameters
  • MemberId: Get from MemberLogin.php API
  • LoginToken: Get from MemberLogin.php API
  • MovieId: Get from Movie List API
  • Example: ./BrowserAdd.php?LoginToken=123123123&MovieId=1&MemberId=1
